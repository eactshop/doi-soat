⚙️ Cấu hình Google Sheet URL

Nhập URL deploy từ Google Apps Script. App dùng JSONP để bypass CORS — cần cập nhật Apps Script để hỗ trợ tham số callback.

Hướng dẫn lấy URL:
1. Mở Google Sheet → Extensions → Apps Script
2. Deploy → New deployment → Web app
3. Execute as: Me · Who has access: Anyone
4. Copy URL và paste vào ô trên
